# sources/ — вход экзокортекса (мультиканальный)

Всё сырьё заходит сюда, по каналам. В Base каждый канал — **дропзона**: кладёшь файлы руками. Коннекторы-автоматизации (почта, календарь, авто-лог ИИ-сессий) — расширения уровней Pro+.

| Канал | Что класть |
|-------|-----------|
| `inbox/` | документы, статьи, транскрипты, черновики — главный вход флагмана `/review-concepts` |
| `mail/` | выгрузки писем и переписок |
| `calendar/` | выгрузки календаря (события, планы) |
| `team-sessions/` | протоколы и транскрипты встреч с людьми |
| `ai-sessions/` | логи диалогов с ИИ, которые стоит переработать в концепты |

## Как обработать

1. Положи файл в подходящий канал (не уверен — клади в `inbox/`).
2. Запусти `/review-concepts` (можно указать имя файла или вставить текст прямо в диалог).
3. Пройди авторское ревью — карточки появятся в `../concepts/`.

## Заметки

- Содержимое каналов по умолчанию **не коммитится** (см. `.gitignore`) — это приватное сырьё. В репозиторий уходят только продукты ревью (карточки концептов).
- Текст — приоритетный источник: готовый транскрипт лучше повторного прогона аудио.
