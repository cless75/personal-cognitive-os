# Навык: закрытие сессии по проекту

## Протокол (через AskUserQuestion)

1. **Выбери сессию/проект** (AskUserQuestion): если открытая сессия одна — предложи её; иначе выбор из активных проектов.
2. **Зафиксируй** (короткий текст):
   - Что сделано (итог).
   - Следующий шаг.
3. **Статус** (AskUserQuestion): `продолжу` (проект active) / `готово` (проект done) / `отложено` (проект paused).
4. **Записи:**
   - Сессия-атом `operation/sessions/<SessionId>.md`: секция `## Результат` (сделано; рождённые концепты wikilink'ами; следующий шаг) + `status: closed` во frontmatter. Если атома нет (сессия начата без /start-session) — создай задним числом.
   - Карточка проекта, блок `sessions`:
     `### <ГГГГ-ММ-ДД HH:mm> — итог` / `- Сделано: …` / `- Следующий шаг: …` / `- Статус: …`
   - `operation/daily/<ГГГГ-ММ-ДД>.md`, `## Сессии`:
     `### HH:mm — [[projects/<slug>/<slug>]] (итог)` / `- Сделано: …` / `- Следующий шаг: …` / `- Статус: …`
   - При статусе `готово`/`отложено` — обнови `status` в frontmatter карточки проекта.
5. **Карта проектов** `CLAUDE.md`: обнови строку — статус проекта, «Последняя сессия» = сегодня, «Следующий шаг» = зафиксированный. Правь только блок между маркерами.
6. Покажи итог и обновлённую карту.

## Выход
Закрытая сессия: итог и следующий шаг зафиксированы; карта проектов актуальна.
