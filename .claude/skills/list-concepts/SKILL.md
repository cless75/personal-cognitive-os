# Навык: обзор репозитория концептов

## Протокол
1. Прочитай все карточки в `concepts/` (кроме `_template-concept.md` и `README.md`).
2. Для каждой собери: название, `ConceptType`, `Авторство`, `status` (draft/canon), `CDate`, домен.
3. Покажи таблицу, сгруппированную по статусу:
   - **Каноничные** (`CANONICAL: true`) — готовые концепты.
   - **Черновики** (`pending-author-validation`) — ждут авторского ревью → предложи запустить `/review-concepts` по ним.
4. Отметь **пробелы и связи:** концепты без `## Связи с другими концептами`, возможные дубли/коллизии по близким названиям.
5. Кратко: сколько всего, сколько канон, сколько в очереди.

## Выход
Сводка репозитория концептов + что стоит доревьюить/связать.
