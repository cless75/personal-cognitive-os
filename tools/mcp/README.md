# tools/mcp/ — конфигурации MCP

Дом для конфигов MCP-серверов (принцип мембраны: интеграции с внешним миром — здесь, а не разбросаны по агентским папкам).

- Конфиг — файл `<server>.json` + короткая заметка `<server>.md` (что даёт, какие секреты нужны).
- **Секреты не коммитятся**: значения — в локальных env/файлах вне git; здесь только структура и плейсхолдеры.
- Подключение к конкретному агенту — через проекции (`sync-agents` разносит по форматам агентов): Claude Code — `~/.claude.json`/`.mcp.json` (JSON), **Codex — `.codex/config.toml` секции `[mcp_servers.*]` (TOML)**.

В Base каталог обычно пуст — MCP-интеграции появляются по мере надобности.
