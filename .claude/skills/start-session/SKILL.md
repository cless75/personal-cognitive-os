# Навык: старт сессии по проекту

## Протокол (через AskUserQuestion)

1. **Выбери проект** (AskUserQuestion): подтяни список из карты проектов `CLAUDE.md` (или из `projects/*/`), предложи выбором.
2. **Цель сессии** (короткий текст): что хочешь сделать за эту сессию.
3. **Отметки:**
   - В карточке проекта `projects/<slug>/<slug>.md`, внутри `<!-- sessions:start -->…<!-- sessions:end -->`, добавь:
     `### <ГГГГ-ММ-ДД HH:mm> — старт`  / `- Цель: <цель>`
   - В `daily/<ГГГГ-ММ-ДД>.md` (создай из шаблона в `daily/README.md`, если нет), в `## Сессии`:
     `### HH:mm — [[projects/<slug>/<slug>]] (старт)` / `- Цель: <цель>`
4. **Карта проектов** `CLAUDE.md`: обнови строку проекта — статус `active`, «Последняя сессия» = сегодня, «Следующий шаг» = цель. Правь только блок между маркерами.
5. Подтверди старт и держи цель сессии в фокусе.

## Выход
Открытая сессия: отметки в карточке проекта, `daily/`, карте проектов.
