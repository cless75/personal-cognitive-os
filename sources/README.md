# sources/ — положи свой файл сюда

Drop-zone для сырья. Сюда кладёшь то, из чего хочешь извлечь концепты:

- документ (`.md`, `.txt`) — статья, заметки, черновик, выгрузка;
- транскрипт сессии/встречи/доклада (текст приоритетнее аудио).

## Как обработать

1. Положи файл в эту папку.
2. Запусти `/review-concepts` (можно указать имя файла или вставить текст прямо в диалог).
3. Пройди авторское ревью — карточки появятся в `../concepts/`.

## Заметки

- Файлы в `sources/` по умолчанию **не коммитятся** (см. `.gitignore`) — это твоё приватное сырьё. В репозиторий уходят только продукты ревью (карточки концептов).
- Текст — приоритетный источник. Если есть готовый транскрипт, не нужно прогонять аудио заново.
