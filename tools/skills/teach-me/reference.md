# teach-me · справка: модель обучения экзокортексу

> Контекст для навыка `teach-me`. Полная модель — в проекте-каноне (707-a6). Здесь — минимум, нужный для ведения диалога в Ex-Base.

## Такты обучения (Ex-Base = такты 0–3)

Обучение = движение по зонам, каждая получает «первое действие». Ex-Base раскрывает **режим Исполнение** (⚙️); режимы Диагностика/Наставник/Партнёр — на уровне Ex-Pro.

| Такт | Действие | Зона | Доказательство ценности |
|------|----------|------|--------------------------|
| **0 «Развернул»** | `/init-exocortex` + `/init-me` | About-Me | «Оно моё и работает» |
| **1 «Первый концепт»** | материал → `/review-concepts` → карточка | sources → concepts | «Из моего материала — моё знание» |
| **2 «Ритм»** | `/start-session` / `/close-session`, журнал | operation (sessions+daily) | «Ничего не теряется» |
| **3 «Мои дела внутри»** | завести проект + область | projects + areas | «Экзокортекс про мою жизнь, не про заметки» |

Такты 4–8 (диагностика → наставник → партнёр → команда → продукт) — за пределами Ex-Base; упоминать как горизонт, не учить здесь.

## Принципы обучения (анти-атрофия)

1. **Ценность в первый час** — довести до такта 1 (первый концепт) как можно раньше.
2. **Desirable difficulties** — гипотеза ученика прежде ответа ИИ.
3. **HITL как тренировка суждения** — авторское ревью, вердикт за учеником, не «ИИ решил».
4. **Один новый смысл за ход** — не грузить всё сразу.
5. **Красная линия:** тренировать суждение и понимание, не подменять их. Цель — чтобы ученик мыслил лучше, а не чтобы ИИ мыслил за него.

## Три канала обучения (контекст)

- **Self-paced в репо** — этот навык `teach-me` + встроенный онбординг; работает офлайн.
- **Школа / когорта (506)** — воркшопы, групповая динамика; драйвер перехода Base→Pro.
- **Менторский трек** — персональная траектория по baseline; ментор ведёт по режимам (Pro+).

`teach-me` обслуживает прежде всего **self-paced**, но пригоден и как материал ментора.

## Карта концепций → такты

K1 (экзокортекс) и K4 (concepts) — такт 1; K5 (operation) — такт 2; K3/K7 (зоны, About-Me≠Areas) — такт 3; K6/K8/K9/K2 — сшивка понимания поверх. Порядок тура — в [concepts.md](concepts.md) «Карта тактов».
