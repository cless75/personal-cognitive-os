# Навык: создание проекта

Заводит проект в портативном формате (чистый Markdown) и регистрирует его в карте проектов.

## Протокол (через AskUserQuestion)

1. **Собери параметры** (AskUserQuestion, где уместно выбор; имя — текстом):
   - Название проекта.
   - Тип/статус: `active` / `paused` (по умолчанию `active`).
   - Короткая цель (одна фраза).
2. **Создай** `projects/<slug>/<slug>.md` из `projects/_template-project.md`:
   - `slug` = kebab-case от названия.
   - Заполни frontmatter (`id`, `title`, `status`, `created: <сегодня>`) и `## Цель`.
3. **Обнови карту проектов** в `CLAUDE.md`: внутри блока `<!-- projects-map:start -->…<!-- projects-map:end -->` добавь строку таблицы:
   `| [[projects/<slug>/<slug>|<title>]] | <status> | — | — |`
   Правь **только** содержимое между маркерами. Если в блоке стоит строка-заглушка «пусто» — замени её.
4. Покажи созданную карточку и обновлённую карту.

## Выход
Папка проекта + карточка + строка в карте проектов `CLAUDE.md`.
