# projects/ — твои проекты

Минимальное, портативное управление проектами. Никаких плагинов, Dataview, счётчиков — чистый Markdown.

## Конвенция

- **Одна папка = один проект:** `projects/<slug>/<slug>.md` (+ вложенные файлы проекта при желании).
- Карточка проекта — плоский frontmatter (`id`, `title`, `status`, `created`) + секции `## Цель`, `## Сессии`, `## Заметки`.
- Шаблон — [`_template-project.md`](_template-project.md).

## Навыки (через выбор вариантов)

| Команда | Что делает |
|---|---|
| `/new-project` | Создать проект + добавить строку в карту проектов `CLAUDE.md` |
| `/start-session` | Начать сессию: выбрать проект и цель, отметка в карточке и `daily/` |
| `/close-session` | Закрыть сессию: итог + следующий шаг, обновить статус в карте |

## Карта проектов

Живёт в корневом [`CLAUDE.md`](../CLAUDE.md) (секция «Карта проектов»), обновляется навыками автоматически. Claude видит её в каждой новой сессии — не нужно пересказывать, где что.
