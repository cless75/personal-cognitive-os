# Навык: закрытие сессии по проекту

## Протокол (через AskUserQuestion)

1. **Выбери сессию/проект** (AskUserQuestion): если открытая сессия одна — предложи её; иначе выбор из активных проектов.
2. **Зафиксируй** (короткий текст):
   - Что сделано (итог).
   - Следующий шаг.
3. **Статус** (AskUserQuestion): `продолжу` (проект active) / `готово` (проект done) / `отложено` (проект paused).
4. **Консолидация из сессии:** если родились находки/различения — предложи прогнать их через `/review-concepts` (концепт получает `source_session:`); проставь в атоме `produced:` — список рождённых карточек (или оставь `[]`).
5. **Финализация атома = Session Note** (`operation/sessions/S…md`; если атома нет — создай задним числом из шаблона README):
   - Секции: `TL;DR` (одна строка-зеркало итога + эмодзи) · `Создано / Обновлено` (wikilinks) · `Решения зафиксированы` · `Следующий шаг`.
   - Frontmatter: `Status:` → `"🟢 Выполнена"` / `"⚪ Отложена"` / `"🔴 Не сработало"`; `Active: false`.
6. **Записи:**
   - Карточка проекта, блок `sessions`:
     `### <ГГГГ-ММ-ДД HH:mm> — итог` / `- Сделано: …` / `- Следующий шаг: …` / `- Статус: …`
   - `operation/daily/<ГГГГ-ММ-ДД>.md`, `## Сессии`:
     `### HH:mm — [[projects/<slug>/<slug>]] (итог)` / `- Сделано: …` / `- Следующий шаг: …` / `- Статус: …`
   - При статусе `готово`/`отложено` — обнови `status` в frontmatter карточки проекта.
7. **Карта проектов** `CLAUDE.md`: обнови строку — статус проекта, «Последняя сессия» = сегодня, «Следующий шаг» = зафиксированный. Правь только блок между маркерами.
8. Покажи итог и обновлённую карту.

## Выход
Закрытая сессия: атом финализирован (TL;DR, `produced:`, `Active: false`); карта проектов актуальна.
