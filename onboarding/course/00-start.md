# Онбординг-курс — карта

Входная дверь универсального онбординг-курса Cognitive OS. Курс учит **двум вещам сразу**: как учиться вместе с ИИ и как жить в собственном экзокортексе. Ядро универсально — конкретную методологию (BoK) подставит твой курс через «бридж» (см. ниже).

Ты проходишь курс в **агентном режиме**: у тебя клон этого репозитория, ты работаешь через Claude Code / Cursor / Cowork и выполняешь практику реальными командами репо (`/init-exocortex`, `/review-concepts`, `/sync-agents` …). Веб-платформа — второй, необязательный режим (обзор в M3).

## Два столпа

| Столп | О чём | Модули |
|---|---|---|
| **A · Принципы обучения с ИИ** | как относиться к ИИ, чтобы он усиливал мышление, а не подменял его | M1, M2 |
| **B · Работа на платформе** | где живёт знание и как с ним работать (место, режимы, BoK) | M0, M3, M4, M5 |

Столпы переплетены: сначала обустраиваешь дом (M0), понимаешь зачем он (M1) и как мыслить с ИИ (M2), затем выбираешь режим работы — веб (M3) или агент (M4) — и наполняешь дом знанием (M5).

## Роль онбординг-скилла (intake)

Курс не начинается с «читай M0 по порядку». Сначала — **intake** (навык-онбординг):

1. **About-Me** — кто ты, роль, контекст. Заполняется через `/init-me` в `About-Me/about-me.md`.
2. **JTBD** — какую работу ты «нанимаешь» этот курс сделать (что должно измениться в твоей практике).
3. **Маршрут** — из About-Me + JTBD скилл собирает твой порядок модулей: кому-то нужен весь путь M0→M5, кому-то хватит связки M1+M2+M5.

Intake — это диагностика на входе (режим 🔍 из M2), а не формальность: он экономит время, отсекая то, что тебе сейчас не нужно.

## «Бридж» к конкретному курсу

Это ядро **универсально** и намеренно не называет ни одной конкретной методологии. Конкретный BoK, который ты подключишь в M5, задаёт **твой курс** — через «бридж»: адрес методологии + точку входа. Один и тот же механизм `/link-bok` подключает любой BoK; курс лишь решает, **какой**. В M5 мы всегда говорим «BoK, который подключит твой курс».

## Команды курса (агентный режим)

| Команда | Где в курсе | Что делает |
|---|---|---|
| `/init-exocortex` | M4 | развернуть/дополнить репо (7 зон + `.exocortex/`) |
| `/init-me` | intake, M4 | заполнить профиль `About-Me/about-me.md` |
| `/review-concepts` | M2, M5 | материал → авторское ревью → карточка концепта в `concepts/` |
| `/link-bok` | M5 | подключить BoK курса как read-only submodule |
| `/update-boks` | M5 | подтянуть обновления подключённых BoK |
| `/sync-agents` | M4 | пересобрать проекции `.claude/`/`AGENTS.md` из канона `tools/skills/` |
