# tools/bok-adaptation/ — внешнее знание на адаптации

Верстак для чужих моделей и методологий, которые ты **перевариваешь под себя**, прежде чем они станут твоими концептами.

- Сюда: конспекты книг/курсов, чужие фреймворки, выдержки внешних BoK — с пометкой источника (`borrowed-from:`).
- Отсюда: после адаптации ядро смысла уходит карточкой в `../../concepts/` (через `/review-concepts`), а исходник остаётся здесь или удаляется.
- Отличие от `boks/` (Pro+): `boks/` — read-only подписки, обновляются платформой; здесь — твоя рабочая переработка.
