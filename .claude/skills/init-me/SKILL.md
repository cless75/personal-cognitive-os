# Навык: инициализация профиля (About-Me)

Заполняет `About-Me/about-me.md` — контекст о владельце, который Claude учитывает во всех сессиях (особенно в авторском ревью — чтобы узнавать голос и присваивать концепты правильно).

## Вход
- Файл владельца (резюме, bio, «обо мне»), путь или вставленный текст — **или** короткий диалог.

## Протокол
1. Если дан файл — извлеки из него: имя/как обращаться, роль, области экспертизы, текущие проекты, стиль/характерные обороты, ценности.
2. Чего не хватает — **спроси коротко** (не длинная анкета; 3–5 вопросов максимум).
3. Заполни секции `About-Me/about-me.md`, сохранив авторские формулировки. Не выдумывай факты — оставляй «…» там, где владелец не ответил.
4. Обнови `updated` в frontmatter на сегодняшнюю дату.
5. Покажи итог и спроси, что уточнить.

## Заметка о приватности
`About-Me/` — приватный профиль. Напомни: в публичный форк его можно не включать (см. `.gitignore`, строка про `/About-Me/`).

## Выход
Заполненный `About-Me/about-me.md` без остаточных токенов `{{...}}`.
