# Навык: sync-agents — проекции канона для агентов

**Инвариант:** один экзокортекс — много агентов. Канон навыков живёт в `tools/skills/` (agent-neutral SKILL.md). Всё, что видят конкретные агенты (`.claude/`, `.claude-plugin/`, `AGENTS.md`, `.cursor/`), — **генерируемые проекции**: их не редактируют руками, их перегенерирует этот навык. Правила маппинга — `.exocortex/projections/README.md`.

## Вход

- `.exocortex/manifest.yaml`: `agents:` (список) + `tier:`.
- `tools/skills/*/SKILL.md` (+ ресурсы навыков) — канон.

## Шаги

1. **Манифест.** Нет `agents:` → спроси (AskUserQuestion, multi-select: claude-code / cowork / cursor / codex / другой) и запиши.
2. **Фильтр по tier:** навык с полем `tier:` выше текущего уровня манифеста в проекции не попадает.
3. **Проекции по агентам:**
   - `claude-code`: скопируй каждый канон-навык в `.claude/skills/<name>/` (SKILL.md + ресурсы), добавив во frontmatter `x_generated: "sync-agents — правь канон в tools/skills/"`; обнови `.claude/commands/*.md` (тонкие вызовы навыков); hooks (`.claude/hooks/`) не трогай — они seed-managed.
   - `cowork`: перегенерируй `.claude-plugin/plugin.json` — список путей `.claude/skills/*` (кроме user-level `init-exocortex`) + `"commands": ".claude/commands"`.
   - `cursor` / `codex` / другой: убедись, что `AGENTS.md` существует и актуален (карта зон + список навыков + правило «канон в tools/skills»); для cursor дополнительно `.cursor/rules` по шаблону из `.exocortex/projections/` (если задан).
4. **Тонкие входы:** проверь `CLAUDE.md` и `AGENTS.md` — заголовок-маркер `generated by sync-agents`, карта зон, список навыков соответствуют канону. **Блок `<!-- projects-map:start/end -->` переноси как есть — он user-owned.**
5. **Идемпотентность:** повторный прогон без изменений канона не создаёт diff. Ничего не пиши в user-owned зоны (About-Me, concepts, operation, sources, projects, areas).
6. **Отчёт:** таблица агент → сгенерировано/без изменений; напомни закоммитить. Удаление агента из `agents:` → предложи удалить его проекцию (только с подтверждением).

## Выход

Актуальные проекции для всех агентов манифеста; репозиторий готов к работе из любого из них.
