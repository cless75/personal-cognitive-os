# areas/ — направления жизни и деятельности

Область — долгоживущее направление **без даты окончания** (здоровье, преподавание, продукт X, семья, финансы). Отличие от проекта: у проекта есть результат и конец, у области — стандарт, который поддерживаешь.

- Одна папка (или один файл) = одна область; шаблон — `_template-area.md`.
- Проекты внутри области ссылаются на неё из своей карточки (`area:` во frontmatter).
- `About-Me/` — **не область**: это профиль владельца, он живёт в корне.

Начни с 1–2 областей, где реально идёт работа — пустой каталог областей хуже отсутствующего.
